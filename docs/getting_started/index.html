Getting Started

If you do not know what Antlir is, start with Overview.

Outside of FB, proceed to Installation. This section assumes that you have a working Antlir in your repo. To check, run buck test //antlir/rpm:test-yum-dnf-from-snapshot-shadowed.

A common Antlir workflow is:

To get started with building images, you may want to study the pages under Concepts & Designs, or take the plunge and try the tutorial on Defining an Image. In both cases, you will want to refer to the Image API as you go.
